# Curso de Verano 2026

Portal de progreso y aprendizaje del Curso de Verano de Poblado Tres.

## Propósito

Ayudar a las familias a comprender:

1. qué actividades realizó el estudiante;
2. qué habilidades practicó;
3. cómo está progresando;
4. cómo pueden acompañarlo en casa.

La portada y la demostración pública utilizan únicamente datos ficticios. Los
expedientes reales fueron retirados de la experiencia pública y deberán
incorporarse en una fase posterior mediante autenticación y permisos por
familia.

## Contenido público

- Presentación del programa para familias
- Demostración de un perfil de progreso ficticio
- Ruta de comprensión lectora
- Actividades de matemáticas, inglés y ciencia
- Cuestionarios de práctica
- Principios de privacidad y trato digno

## Escala descriptiva

- Iniciando
- En desarrollo
- Avance consistente
- Logro observado

Estos estados no son calificaciones oficiales ni diagnósticos clínicos.

## Pendiente antes de utilizar datos reales

- Consentimiento y aviso de privacidad completo
- Acceso autenticado para docente y familias
- Base de datos y copias de seguridad
- Permisos para evidencias y fotografías
- Política de conservación y eliminación
- Registro central de actividades y observaciones

## Evaluación formativa

Los accesos vigentes utilizan `evaluaciones/practica-formativa.html`.

- Un error no borra el avance.
- El primer error presenta una pista.
- El segundo intento incorrecto muestra la respuesta y una explicación.
- El reporte distingue logro al primer intento, uso de pista y revisión.
- El resultado se expresa con la escala descriptiva del portal.
- Solo se solicita nombre de pila o código; el resultado no se guarda en línea.

`modelo-seguimiento.json` documenta las entidades mínimas para la futura base de
datos privada. No contiene información real de estudiantes.

## Tecnología actual

HTML, CSS y JavaScript. La demostración funciona como sitio estático.
